About this example

This example shows the numberBox dat.GUI controller for numeric values (UI: input field; API: .addAs(object, "member", "numberBox")). This example also showcases the .min(), .max() and .step() configuration APIs on a regular number dat.GUI controller attached to the third object member: z.
